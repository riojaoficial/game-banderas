# 🌍 World Cup 2026 Flag Guessing Game

A fun and interactive flag guessing game featuring all 48 countries qualified for the FIFA World Cup 2026!

## 🎮 Game Features

- **All 48 World Cup 2026 Teams**: Quiz yourself with every country participating in the next championship
- **Interactive Gameplay**: Multiple-choice questions with immediate feedback
- **Hints System**: Get helpful hints for each country
- **Skip Option**: Skip questions you find difficult
- **Score Tracking**: Real-time score, accuracy percentage, and performance metrics
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Beautiful UI**: Modern gradient design with smooth animations

## 🚀 Getting Started

### Requirements
- Web browser with JavaScript enabled
- No installation needed!

### How to Play
1. Open `index.html` in your web browser
2. A flag emoji will be displayed on screen
3. Select the correct country name from the 4 options
4. Use hints if you need help
5. Skip questions if you want to move on
6. Try to get the highest accuracy possible!
7. Complete all 48 questions to see your final score

## 🎯 Game Mechanics

### Scoring
- **Correct Answer**: +1 point
- **Skip/Wrong**: No points gained

### Features
- **Real-time Score**: Track your points as you play
- **Accuracy Percentage**: See how well you're doing
- **Hint System**: Get geographical clues about each country (one per question)
- **Skip Button**: Move to the next question without answering
- **Performance Rating**: Get feedback on your overall performance

### Performance Levels
- 🏆 **100%**: Perfect! You're a geography expert!
- 🥇 **90%+**: Excellent! Outstanding knowledge!
- 🥈 **75-89%**: Great! Very good performance!
- 🥉 **60-74%**: Good! Keep practicing!
- 👍 **40-59%**: Not bad! Learn more countries!
- 📚 **Below 40%**: Keep learning! You'll improve!
